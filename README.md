# Mu

**Early Stages. Work in progress.**

Mu is a work in process text-based MMORPG server written in elixir and built on the Kalevala MUD framework.

# Setup

```
mix deps.get
iex -S mix
```

# Login
Login with any MUD client such as Mudlet, TinTin++, or Blightmud. 

Host: `127.0.0.1`
Port: `4444`
